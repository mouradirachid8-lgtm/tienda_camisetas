# Tienda de camisetas


## Guía temporal (Voy a buscar otras formas mientras tanto)
### Para ejecutar el proyecto desde 0, seguid los siguientes pasos:
1. Clonar el repositorio
2. Añadir el archivo .env que se encuentra en la página principal del repositorio en el directorio del proyecto.
3. Ejecutar el siguiente comando: 
```
composer install
```
4. Luego ejecutar el siguiente comando para instalar dependencias:
```
php artisan key:generate
```
5. Para ejecutar el proyecto. Usad el siguiente comando:
```
php artisan serve
```

</br>

## Tras esto, el proyecto debería estar ejecutándose en [http://127.0.0.1:8000](http://127.0.0.1:8000)
